Sample Applications

The followings are sample applications that demonstrate how to use SIP servlet technology to build converged SIP/HTTP/JSR-309/XMPP applications based on Voxeo Prism Application Server.


Sample Application Directory Brief Description
A Prism Demo Application prism This sample demonstrates how to write a simple application with SIP Servlet API and JSR-309 API
An Echo Application echo This application demonstrates how a simple user agent server handles SIP INVITE and MESSAGE messages.
A Back-To-Back User Agent b2bua This sample demonstrates how to write a simple B2BUA SIP servlet.
A Find-Me Application findme This sample demonstrates how to write a simple Find-Me and Single Line Extension application by leveraging built-in SIP proxy function. This sample also demonstrate how to write a JWS based Web Service.
A Click-To-Dial Application clicktodial This sample demonstrates how to write a simple Click-To-Dial application. It also demonstrate how to use Web Services to do Click-To-Dial.
An IVR application vxml This sample demonstrates how to write a simple IVR Application.
An Application Router ar This sample demonstrates how to write a simple Application Router.
SIPMethod API Sample extension This sample demonstrates how to use SIPMethod APIs.
A 3-way Conference Application userjoin This is a sample conferencing application based on Third Party Joins.
A Convergence Application convergence It demonstrates how SIP, HTTP and XMPP servlets can work together.
A Simple XMPP Server Application xmppecho demonstrates how a simple XMPP IM server handles XMPP client logon, client messages, XMPP s2s communication, and s2s messages..
A Simple XMPP Client Application xmppclient This sample application demonstrates how to implement a XMPP client using XMPP servlet technology.
A Player Application play This sample demonstrates how to write a simple play application with JSR-309 API.
A Aynchronized Player Application asyncPlay This sample demonstrates how to write a simple play application that uses asynchronous join operation with JSR-309 API
A Prompt and Collect Application collect This sample demonstrates how to write a prompt and collect application with JSR-309 API
A Minimal Conference Application miniconf This sample demonstrates how to write a simple single conference application with JSR-309 API
A Multiparty Conference Application multiparty This sample demonstrates how to write a simple multiparty conference application with JSR-309 API
A Voicemail Application voicemail This sample demonstrates how to write a simple voicemail application with JSR-309 API
A Registrar Application registrar This sample demonstrates a simple registrar.
A Simple Distributed Multiparty Application dmultiparty This sample demonstrates how to write a simple distributed multiparty application with JSR-309 API and Prism API.
OSGi Sample dmultiparty This sample application demonstrates how to write a application act as a OSGi bundle. It also demonstrates how to easily using the OSGi service.
